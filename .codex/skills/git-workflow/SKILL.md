# Git Workflow Standards

Before committing:

- Run tests
- Run lint
- Check formatting

Commit message format:

- feat:
- fix:
- refactor:
- chore:
- docs:

Examples:

- feat: add react query caching for study sets
- fix: resolve SignalR reconnection issue
- refactor: simplify auth middleware

After changes:

- Create concise summary
- Suggest PR title
