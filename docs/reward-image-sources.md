# Reward Image Sources

The reward catalog images are downloaded and bundled locally in `assets/images/rewards/` so the app can display them without fetching images at runtime.

| Asset | Source |
| --- | --- |
| `eco-bag.png` | Wikimedia Commons, `Blue reusable shopping bag.JPG` |
| `rice.png` | Wikimedia Commons, `A bowl of rice.jpg` |
| `canned-goods.png` | Pexels, Julia M Cameron, `A Close-Up Shot of Canned Goods` |
| `gcash.png` | Wikimedia Commons, `GCash logo.svg` |
| `school-supplies.png` | Pixabay, `Back To School School Supplies` |

These files are resized/cropped to 320x320 PNGs for consistent display in the Rewards screen.
